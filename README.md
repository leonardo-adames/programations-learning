# programations-learning

Mi nombre es Leonardo José Marte Adames, estudiante de programación, Bases De Datos y Desarrollo Web, Creo este repositorio con fines educativos para documentar mi curva de aprendizaje en tiempo real y compartir  este proceso y toda la información y conocimientos compartidos por esta vía, que Dios les bendiga a todos los visitante de este perfil

Este repositorio documenta mi aprendizaje general de programación.  
Incluye fundamentos, lógica, estructuras de control, algoritmos básicos y ejercicios diseñados para fortalecer mi pensamiento computacional.

---

## 📚 Estructura del repositorio

- **notes/** → Apuntes, teoría, conceptos clave y resúmenes de programación en general  
- **exercises/** → Ejercicios de lógica, pseudocódigo y pequeños programas  
- **projects/** → Mini proyectos que integran varios conceptos de programación  
- **challenges/** → Retos de dificultad progresiva (algoritmos, problemas clásicos, etc.)  
- **examples/** → Ejemplos de código y patrones básicos

---

## 🎯 Objetivos del aprendizaje

- Desarrollar una base sólida de lógica de programación  
- Comprender las estructuras de control y tipos de datos  
- Practicar con ejercicios y problemas clásicos  
- Construir pequeños proyectos que refuercen los conceptos  
- Crear una base conceptual que apoye otros lenguajes y tecnologías
